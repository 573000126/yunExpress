使用方法:

1、服务端代码,桌面/控制台等程序，可以直接添加服务引用

        RefAPI.TNA.TrackNumberAssignClient client = new RefAPI.TNA.TrackNumberAssignClient();
            var r = client.TrackNumberAssignStandard(15, 1, "");
            Console.WriteLine(r.IsSuccess);
            Console.WriteLine(r.ErrorMessage);
            Console.WriteLine(r.Data.Count());
            Console.ReadKey();

2、网站可以直接发送HTTP请求,返回JSON数据格式 

      请求:  http://localhost:16103/TrackNumberAssign.svc/TrackNumberAssignStandard?shippingMethodId=15&count=1 
 
        http://211.154.151.46:836/TrackNumberAssign.svc/TrackNumberAssignStandard?shippingMethodId=15&count=1 
      返回:
      
            {"Data":["TESTZXQ00001"],"ErrorMessage":null,"IsSuccess":true}
